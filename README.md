# English Learning System

Приложение для изучения слов и предложений (перевод RU→EN). Данные добавляются только из JSON-файла; взаимодействие с ИИ происходит в другом сервисе.

## Как пользоваться

**Десктоп (Linux):** таймер помодоро + тренажёр перевода по текущей порции.
```bash
python3 run.py app
```
(Если нет tkinter: `sudo apt install python3-tk`.)

**Консоль — только эти команды:**
```bash
python3 run.py                    # меню
python3 run.py app                # открыть приложение
python3 run.py import-json file.json   # добавить слова/предложения из JSON в текущую порцию
python3 run.py list-portions      # список порций (id, предложений, слов)
python3 run.py set-portion <id>   # переключиться на порцию
python3 run.py next-portion       # создать новую порцию и переключиться на неё
python3 run.py generate-audio    # сгенерировать аудио + HTML для телефона (текущая порция)
```

## Порции

Данные хранятся **порциями** (пакетами). Одна порция — один список предложений и слов. Ты сам решаешь, когда перейти к следующей порции (кнопка «Следующая порция» или `next-portion`). Добавление данных — только через **import-json** (файл готовишь сам или получаешь из другого сервиса).

Формат JSON: `{"sentences": [...], "words": [...]}` — каждый элемент с полями `english`/`en` и `translation`/`ru`.

## Приложение

- **Таймер:** 25 мин / 5 мин. При старте воспроизводится `data/sounds/start.wav`, по окончании — `data/sounds/end.wav` (если файлов нет — системный beep).
- **Тренажёр:** показывается русское слово/предложение, вводишь английский. Есть кнопка **«Посмотреть ответ»** — если ею воспользовался, этот пункт не идёт в процент успешных (процент только для информации).
- **Ошибки:** если написал с ошибкой, под блоком результата появляется кнопка **«Записать ошибку»** — в БД сохраняется правильный и неправильный вариант (без типа ошибки).
- **Порции:** в шапке — «Сменить порцию» (выбор из списка) и «Следующая порция» (новая пустая порция).
- Ошибка **только в знаках препинания** не считается ошибкой.

## Генерация аудио

Аудио создаётся через **Google Cloud Text-to-Speech** (американский английский, голос en-US-Wavenet-D), к каждой записи добавляется 5 сек тишины. Нужны: учётная запись Google Cloud, включённый API Text-to-Speech, `pip install google-cloud-texttospeech`, переменная окружения `GOOGLE_APPLICATION_CREDENTIALS` с путём к ключу сервисного аккаунта. HTML для просмотра на телефоне (EN + кнопка «Показать перевод») пишется в `data/audio/review_sentences.html` и `review_words.html`.
